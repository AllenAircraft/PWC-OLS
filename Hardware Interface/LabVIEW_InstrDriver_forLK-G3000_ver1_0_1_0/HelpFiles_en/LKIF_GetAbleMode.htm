Get ABLE (LKIF_GetAbleMode)

BOOL WINAPI LKIF_GetAbleMode(IN int HeadNo,OUT LKIF_ABLEMODE *AbleMode);
Description Returns the ABLE set value.
Parameter HeadNo Sets the returned head number. Sets 0 (HEAD-A) or 1 (HEAD-B).
AbleMode Sets the mode.
Returned Value TRUE Completes without errors
FALSE Completes with errors
Reference Refer to Set ABLE for more about LKIF_ABLEMODE.
