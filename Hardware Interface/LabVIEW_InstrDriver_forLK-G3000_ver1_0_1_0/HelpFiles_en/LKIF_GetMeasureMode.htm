Get Measurement Mode (LKIF_GetMeasureMode)

BOOL WINAPI LKIF_GetMeasureMode(IN int HeadNo,OUT LKIF_MEASUREMODE *MeasureMode);
Description Returns the measurement mode.
Parameter HeadNo Sets the returned head number. Sets 0 (HEAD-A) or 1 (HEAD-B).
MeasureMode Returns the measurement mode.
Returned Value TRUE Completes without errors
FALSE Completes with errors
Reference Refer to Set Measurement Mode for more about LKIF_MEASUREMODE.
