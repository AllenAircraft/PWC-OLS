Get Data Storage (LKIF_GetDataStorage)

BOOL WINAPI LKIF_GetDataStorage(OUT LKIF_TARGETOUT *TargetOut,OUT int *NumStorage,OUT LKIF_STORAGECYCLE *StorageCycle);
Description Returns the target OUT, number of accumulations, and accumulation cycle of the data storage settings.
Parameter TargetOut Returns the target OUT.
NumStorage Returns the number of accumulations.
StorageCycle Returns the accumulation cycle.
Returned Value TRUE Completes without errors
FALSE Completes with errors
Reference Refer to Set Data Storage for more about LKIF_TARGETOUT and LKIF_STORAGECYCLE.
