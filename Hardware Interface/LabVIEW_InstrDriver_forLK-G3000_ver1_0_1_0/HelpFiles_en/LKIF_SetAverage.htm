Set Number of Times for Averaging (LKIF_SetAverage)

BOOL WINAPI LKIF_SetAverage(IN int OutNo,IN LKIF_AVERAGE Average);
Description Sets the number of times for averaging.
Parameter OutNo The OUT number to be set. Sets 0 (OUT1) or 1 (OUT2).
Average Number of times for averaging. The following values can be set.

typedef enum {
    LKIF_AVERAGE_1, // 1 time
    LKIF_AVERAGE_4, //
    LKIF_AVERAGE_16, //
    LKIF_AVERAGE_64, //
    LKIF_AVERAGE_256, //
    LKIF_AVERAGE_1024, //
    LKIF_AVERAGE_4096, //
    LKIF_AVERAGE_16384, //
    LKIF_AVERAGE_65536, //
    LKIF_AVERAGE_262144, // 262144 times
} LKIF_AVERAGE;
Returned Value TRUE Completes without errors
FALSE Completes with errors
