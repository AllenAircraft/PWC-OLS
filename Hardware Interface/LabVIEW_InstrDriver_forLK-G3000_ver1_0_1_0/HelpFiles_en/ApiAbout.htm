File composition

LkIF.dll DLL application.
KeyUsbDrv.dll DLL needed to Communicate with USB.
LkIF.h Header file that defines API. Use this when creating files in C/C++.
LkIF.BAS Header file that defines API. Use this when creating files in Visual Basic.
Samples/VB Sample source folder.Contains a sample program created with Visual Basic.
HelpFiles This file.

This DLL checks operations in Visual C++ 6.0 or Visual Basic 6.0.

Install the USB driver as a part of LK-Navigator.

API composition

This DLL is divided into four basic categories.

Mode Change Command This API switches between "Normal mode" and "Communication mode".
Measurement Control Command This API controls measurement.It exchanges data with the main unit during "Normal mode".
Change Parameter Command This API changes the parameters.It exchanges data with the main unit during "Communication mode".
Check Parameter Command This API obtains the parameters.It exchanges data with the main unit during "Communication mode".
