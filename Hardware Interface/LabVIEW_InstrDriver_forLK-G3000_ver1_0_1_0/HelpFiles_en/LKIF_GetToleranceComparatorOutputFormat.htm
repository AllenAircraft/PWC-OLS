Get Comparator Output Format (LKIF_GetToleranceComparatorOutputFormat)

BOOL WINAPI LKIF_GetToleranceComparatorOutputFormat(OUT LKIF_TOLERANCE_COMPARATOR_OUTPUT_FORMAT *ToleranceComparatorOutputFormat);
Description Returns the comparator output format.
Parameter ToleranceComparatorOutputFormat Returns the comparator output format.
Returned Value TRUE Completes without errors
