Get Number of Times of Alarm Processing (LKIF_GetNumAlarm)

BOOL WINAPI LKIF_GetNumAlarm(IN int HeadNo,OUT int *NumAlarm);
Description Returns the number of times of alarm processing.
Parameter HeadNo Sets 0 (HEAD-A) or 1 (HEAD-B) as the head number.
NumAlarm Returns the number of times of alarm processing. Returns a value from 0 to 999.
Returned Value TRUE Completes without errors
FALSE Completes with errors
