Get Scaling (LKIF_GetScaling)

BOOL WINAPI LKIF_GetScaling(IN int OutNo,IN int HeadNo,OUT int *InputValue1,OUT int *OutputValue1,OUT int *InputValue2,OUT int *OutputValue2);
Description Returns the scaling.
Parameter OutNo The OUT number to be set. Sets 0 (OUT1) or 1 (OUT2).
HeadNo Sets 0 (HEAD-A) or 1 (HEAD-B) as the head number.
InputValue1 Returns the input value 1. Returns a value from -999999 to 99999.
OutputValue1 Returns the display value 1. Returns a value from -999999 to 99999.
InputValue2 Returns the input value 2. Returns a value from -999999 to 99999.
OutputValue2 Returns the display value 2. Returns a value from -999999 to 99999.
Returned Value TRUE Completes without errors
FALSE Completes with errors
Reference The specified values vary depending on the minimum display unit of the equipment.
Ex: When the tolerance is set as 999999, the specified value for the equipment is set as follows.
  Minimum display unit=0.01mm: 9999.99mm
  Minimum display unit=0.001mm: 999.999mm
  Minimum display unit=0.0001mm: 99.9999mm
  Minimum display unit=0.00001mm: 9.99999mm
  Minimum display unit=0.1µm: 99999.9µm
  Minimum display unit=0.1µm: 9999.99µm
