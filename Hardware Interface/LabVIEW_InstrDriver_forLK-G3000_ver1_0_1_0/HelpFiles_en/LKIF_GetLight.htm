Receive Light Waveform (LKIF_GetLight)

BOOL WINAPI LKIF_GetLight(IN int HeadNo,IN int PeekNo,OUT int *MeasurePosition,OUT int *NumReaded,OUT BYTE *Value);
Description Returns the light waveform..
Parameter HeadNo Returned Head number. Sets 0 (HEAD-A) or 1 (HEAD-B).
PeekNo Sets the peak position of the received waveform when measurement mode is set to "transparent object 2". Sets the values 0 to 3.
MeasurePosition Returns the measurement position. Four sequences can be set. The range of the received data is 0 to 1024, where 0 represents invalid data and all other values represent valid measurement position data.
NumReaded Returns the amount of read valid data.
Value Returns the amount of read data. 1024 sequences can be set.
Returned Value TRUE Completes without errors
FALSE Completes with errors
