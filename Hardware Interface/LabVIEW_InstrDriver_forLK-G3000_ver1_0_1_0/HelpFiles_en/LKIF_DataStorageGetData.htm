Outputting the Data Storage(LKIF_DataStorageGetData)

BOOL WINAPI LKIF_DataStorageGetData(IN int OutNo,IN int NumOutBuffer,OUT LKIF_FLOATVALUE *OutBuffer,OUT int *NumReceived);
Description Returns the data storage accumulation.
Parameter OutNo The returned OUT number. Sets 0 (OUT1) or 1 (OUT2).
NumOutBuffer Sets the amount of data to be returned. Sets from 1 to 65536.
OutBuffer Returns the accumulation data.
NumReceived Returns the amount of received accumulation.
Returned Value TRUE Completes without errors
FALSE Completes with errors
Reference Note any returned errors during data storage accumulation.
Refer to Measurement Value Output for more about LKIF_FLOATVALUE.
