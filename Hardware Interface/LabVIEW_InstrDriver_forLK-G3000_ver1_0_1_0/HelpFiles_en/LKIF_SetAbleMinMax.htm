Set ABLE Control Range (LKIF_SetAbleMinMax)

BOOL WINAPI LKIF_SetAbleMinMax(IN int HeadNo,IN int Min,IN int Max);
Description Sets the ABLE control range.
Parameter HeadNo Sets 0 (HEAD-A) or 1 (HEAD-B) as the head number.
Min Sets the minimum ABLE value. Sets the values 1 to 99.
Max Sets the maximum ABLE value. Sets the values 1 to 99.
Returned Value TRUE Completes without errors
FALSE Completes with errors
