Starting the ABLE Calibration (LKIF_AbleStart)

BOOL WINAPI LKIF_AbleStart(IN int HeadNo);
Description Starts the ABLE calibration.
Parameter HeadNo Sets the head number.Sets 0 (HEAD-A) or 1 (HEAD-B).
Returned Value TRUE Completes without errors
FALSE Completes with errors
Note This function finishes with the unit in Communication mode. After calling this function, you must call either Finishing the ABLE Calibration API or Stopping the ABLE Calibration API.
