Set Trigger Mode (LKIF_SetTriggerMode)

BOOL WINAPI LKIF_SetTriggerMode(IN int OutNo,IN LKIF_TRIGGERMODE TriggerMode);
Description Sets the trigger mode.
Parameter OutNo The OUT number to be set. Sets 0 (OUT1) or 1 (OUT2).
TriggerMode Trigger mode. The following values can be set.

typedef enum {
    LKIF_TRIGGERMODE_EXT1, // external trigger 1
    LKIF_TRIGGERMODE_EXT2, // external trigger 2
} LKIF_TRIGGERMODE;
Returned Value TRUE Completes without errors
FALSE Completes with errors
