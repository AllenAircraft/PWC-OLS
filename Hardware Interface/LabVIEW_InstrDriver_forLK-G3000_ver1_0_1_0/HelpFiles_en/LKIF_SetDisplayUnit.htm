Set Minimum Display Unit (LKIF_SetDisplayUnit)

BOOL WINAPI LKIF_SetDisplayUnit(IN int OutNo,IN LKIF_DISPLAYUNIT DisplayUnit);
Description

Sets the minimum display unit.
Parameter OutNo The OUT number to be set. Sets 0 (OUT1) or 1 (OUT2).
DisplayUnit Minimum display unit. The following values can be set.

typedef enum {
    LKIF_DISPLAYUNIT_0000_01MM, // 0.01mm
    LKIF_DISPLAYUNIT_000_001MM, // 0.001mm
    LKIF_DISPLAYUNIT_00_0001MM, // 0.0001mm
    LKIF_DISPLAYUNIT_0_00001MM, // 0.00001mm
    LKIF_DISPLAYUNIT_0000_01UM, // 0.01µm
    LKIF_DISPLAYUNIT_000_001UM, // 0.001µm
} LKIF_DISPLAYUNIT;
Returned Value TRUE Completes without errors
FALSE Completes with errors
