Get Trigger Mode (LKIF_GetTriggerMode)

BOOL WINAPI LKIF_GetTriggerMode(IN int OutNo,OUT LKIF_TRIGGERMODE *TriggerMode);
Description Returns the trigger mode.
Parameter OutNo The OUT number to be set. Sets 0 (OUT1) or 1 (OUT2).
TriggerMode Returns the trigger mode.
Returned Value TRUE Completes without errors
FALSE Completes with errors
Reference Refer to Set Trigger Mode for more about LKIF_TRIGGERMODE.
