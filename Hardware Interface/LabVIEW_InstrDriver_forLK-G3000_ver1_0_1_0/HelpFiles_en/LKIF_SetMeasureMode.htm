Set Measurement Mode (LKIF_SetMeasureMode)

BOOL WINAPI LKIF_SetMeasureMode(IN int HeadNo,IN LKIF_MEASUREMODE MeasureMode);
Description Sets the measurement mode.
Parameter HeadNo Sets the specified head number. Sets 0 (HEAD-A) or 1 (HEAD-B).
MeasureMode Measurement mode. The following values can be set.

typedef enum {
    LKIF_MEASUREMODE_NORMAL, // normal
    LKIF_MEASUREMODE_HALF_T, // translucent object
    LKIF_MEASUREMDOE_TRAN_1, // transparent object
    LKIF_MEASUREMODE_TRAN_2, // transparent object 2
    LKIF_MEASUREMODE_MRS, // multireflective object
} LKIF_MEASUREMODE;
Returned Value TRUE Completes without errors
FALSE Completes with errors
