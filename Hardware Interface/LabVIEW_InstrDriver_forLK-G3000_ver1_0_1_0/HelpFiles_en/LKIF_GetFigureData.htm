Statistical Results Output (LKIF_GetFigureData)

BOOL WINAPI LKIF_GetFigureData(IN int OutNo,OUT LKIF_FIGUREDATA *FigureData);
Description Returns the statistical results.
Parameter OutNo The returned OUT number. Sets 0 (OUT1) or 1 (OUT2).
FigureData Returns the statistical results data. The following structures can be set.

typedef struct {
    float ToleUpper; // tolerance upper limit
    float ToleLower; // tolerance lower limit
    float AverageValue; // average value
    float MaxValue; // maximum value
    float MinValue; // minimum value
    float DifValue; // maximum value ? minimum value
    float SdValue; // standard deviation
    LONG DataCnt; // number of all data
    LONG HighDataCnt; // number of tolerance High data
    LONG GoDataCnt; // number of tolerance Go data
    LONG LowDataCnt; // number of tolerance Low data
} LKIF_FIGUREDATA;
Returned Value TRUE Completes without errors
FALSE Completes with errors
