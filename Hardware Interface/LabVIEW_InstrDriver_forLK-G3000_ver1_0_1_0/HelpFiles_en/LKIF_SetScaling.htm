Set Scaling (LKIF_SetScaling)

BOOL WINAPI LKIF_SetScaling(IN int OutNo,IN int HeadNo,IN int InputValue1,IN int OutputValue1,IN int InputValue2,IN int OutputValue2);
Description Sets the scaling.
Parameter OutNo The OUT number to be set. Sets 0 (OUT1) or 1 (OUT2).
HeadNo The head number to be set. Sets 0 (HEAD-A) or 1 (HEAD-B).
InputValue1, OutputValue1 Sets the value for input value to display value. Sets a value from -999999 to 999999.
InputValue2, OutputValue2 Sets the value for input value to display value. Sets a value from -999999 to 999999.
Returned Value TRUE Completes without errors
FALSE Completes with errors
Reference The parameters input value and output value can be set in the following range.

span = (display value 1 - display value 2) / (input value 1 - input value 2)
-9.999 <= span <= 9.999

The specified values vary depending on the minimum display unit of the equipment.
Ex: When the scaling is set as 999999, the specified value for the equipment is set as follows.
  Minimum display unit=0.01mm: 9999.99mm
  Minimum display unit=0.001mm: 999.999mm
  Minimum display unit=0.0001mm: 99.9999mm
  Minimum display unit=0.00001mm: 9.99999mm
  Minimum display unit=0.1µm: 99999.9µm
  Minimum display unit=0.1µm: 9999.99µm
