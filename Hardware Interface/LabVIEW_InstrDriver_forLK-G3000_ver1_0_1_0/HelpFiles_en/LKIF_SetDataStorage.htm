Set Data Storage (LKIF_SetDataStorage)

BOOL WINAPI LKIF_SetDataStorage(IN LKIF_TARGETOUT TargetOut,,IN int NumStorage,IN LKIF_STORAGECYCLE StorageCycle);
Description Sets the target OUT, number of accumulations, and accumulation cycle of the data storage settings.
Parameter TargetOut Target OUT. The following values can be set.

typedef enum {
    LKIF_TARGETOUT_NONE, // no target OUT
    LKIF_TARGETOUT_OUT1, // OUT‚P
    LKIF_TARGETOUT_OUT2, // OUT‚Q
    LKIF_TARGETOUT_BOTH, // OUT1 and OUT2
} LKIF_TARGETOUT;
NumStorage Sets the number of accumulations. Sets a value from 1 to 65536.
StorageCycle Accumulation cycle. The following values can be set.

typedef enum {
    LKIF_STORAGECYCLE_1, // sampling rate x 1
    LKIF_STORAGECYCLE_2, // sampling rate x 2
    LKIF_STORAGECYCLE_5, // sampling rate x 5
    LKIF_STORAGECYCLE_10, // sampling rate x 10
    LKIF_STORAGECYCLE_20, // sampling rate x 20
    LKIF_STORAGECYCLE_50, // sampling rate x 50
    LKIF_STORAGECYCLE_100, // sampling rate x 100
    LKIF_STORAGECYCLE_200, // sampling rate x 200
    LKIF_STORAGECYCLE_500, // sampling rate x 500
    LKIF_STORAGECYCLE_1000, // sampling rate x 1000
} LKIF_STORAGECYCLE;
Returned Value TRUE Completes without errors
FALSE Completes with errors
