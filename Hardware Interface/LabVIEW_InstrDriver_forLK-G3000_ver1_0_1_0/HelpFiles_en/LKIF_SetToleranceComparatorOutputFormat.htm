Set Comparator Output Format (LKIF_SetToleranceComparatorOutputFormat)

BOOL WINAPI LKIF_SetToleranceComparatorOutputFormat(IN LKIF_TOLERANCE_COMPARATOR_OUTPUT_FORMAT ToleranceComparatorOutputFormat);
Description Sets the comparator output format.
Parameter ToleranceComparatorOutputFormat Comparator output format mode. The following values can be set.

typedef enum {
    LKIF_TOLERANCE_COMPARATOR_OUTPUT_FORMAT_NORMAL, // normal
    LKIF_TOLERANCE_COMPARATOR_OUTPUT_FORMAT_HOLD, // hold
    LKIF_TOLERANCE_COMPARATOR_OUTPUT_FORMAT_OFF_DELAY, // off-delay
} LKIF_TOLERANCE_COMPARATOR_OUTPUT_FORMAT;
Returned Value TRUE Completes without errors
FALSE Completes with errors
