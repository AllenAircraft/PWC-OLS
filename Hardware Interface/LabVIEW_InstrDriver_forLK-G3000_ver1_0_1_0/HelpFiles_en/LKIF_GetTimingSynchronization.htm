Get Timing Synchronization (LKIF_GetTimingSynchronization)

BOOL WINAPI LKIF_GetTimingSynchronization(OUT LKIF_SYNCHRONIZATION *Synchronization);
Description Returns the timing synchronization.
Parameter Synchronization Returns the timing synchronization.
Returned Value TRUE Completes without errors
FALSE Completes with errors
Reference Refer to Set Timing Synchronization for more about LKIF_SYNCHRONIZATION.
