Set Timing Synchronization (LKIF_SetTimingSynchronization)

BOOL WINAPI LKIF_SetTimingSynchronization(IN LKIF_SYNCHRONIZATION Synchronization);
Description Sets the timing synchronization.
Parameter Synchronization Timing synchronization. The following values can be set.

typedef enum {
    LKIF_SYNCHRONIZATION_ASYNCHRONOUS, // asynchronous
    LKIF_SYNCHRONIZATION_SYNCHRONIZED, // synchronous
} LKIF_SYNCHRONIZATION;
Returned Value TRUE Completes without errors
FALSE Completes with errors
