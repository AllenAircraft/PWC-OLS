Get Analog Output Scaling (LKIF_GetAnalogScaling)

BOOL WINAPI LKIF_GetAnalogScaling(IN int OutNo,OUT int *InputValue1,OUT int *OutputVoltage1,OUT int *InputValue2,OUT int *OutputVoltage2);
Description Returns the analog output scaling.
Parameter OutNo The returned OUT number. Sets 0 (OUT1) or 1 (OUT2).
InputValue1 Returns the measurement value 1. Returns a value from -999999 to 999999.
OutputVoltage1 Returns the output voltage 1. Returns a value from -10500 (mV) to 10500 (mV). (Units are set.)
InputValue2 Returns the measurement value 2. Returns a value from -999999 to 999999.
OutputVoltage2 Returns the output voltage 2. Returns a value from -10500 (mV) to 10500 (mV). (Units are set.)
Returned Value TRUE Completes without errors
FALSE Completes with errors
Reference The specified values vary depending on the minimum display unit of the equipment.
Ex: When measurement value 1 is set as 999999, the specified value for the equipment is set as follows.
  Minimum display unit=0.01mm: 9999.99mm
  Minimum display unit=0.001mm: 999.999mm
  Minimum display unit=0.0001mm: 99.9999mm
  Minimum display unit=0.00001mm: 9.99999mm
  Minimum display unit=0.1µm: 99999.9µm
  Minimum display unit=0.1µm: 9999.99µm

The output voltage is returned in mV units.
