Set Analog-Through (LKIF_SetAnalogThrough)

BOOL WINAPI LKIF_SetAnalogThrough(IN int OutNo,IN BOOL IsOn);
Description Sets the alarm through.
Parameter OutNo The OUT number to be set. Sets 0 (OUT1) or 1 (OUT2).
IsOn Sets analog through to ON (TRUE) or OFF (FALSE).
Returned Value TRUE Completes without errors
FALSE Completes with errors
