Set Filter Mode (LKIF_SetFilterMode)

BOOL WINAPI LKIF_SetFilterMode(IN int OutNo,OUT LKIF_FILTERMODE FilterMode);
Description Sets the filter mode.
Parameter OutNo The OUT number to be set. Sets 0 (OUT1) or 1 (OUT2).
FilterMode Sets the filter mode. The following values can be set.

typedef enum {
    LKIF_FILTERMODE_MOVING_AVERAGE, // moving average
    LKIF_FILTERMODE_LOWPASS, // low pass filter
    LKIF_FILTERMODE_HIGHPASS, // high pass filter
} LKIF_FILTERMODE;
Returned Value TRUE Completes without errors
FALSE Completes with errors
