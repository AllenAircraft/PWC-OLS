Get Tolerance (LKIF_GetTolerance)

BOOL WINAPI LKIF_GetTolerance(IN int OutNo,OUT int *UpperLimit,OUT int *LowerLimit,OUT int *Hysteresis);
Description Returns the tolerance.
Parameter OutNo The OUT number to be set. Sets 0 (OUT1) or 1 (OUT2).
UpperLimit Tolerance upper limit. Sets a value from -999999 to 999999.
LowerLimit Tolerance lower limit. Sets a value from -999999 to 999999.
Hysteresis Hysteresis. Sets a value from 0 to 999999.
Returned Value TRUE Completes without errors
FALSE Completes with errors
Reference The specified values vary depending on the minimum display unit of the equipment.
Ex: When the received tolerance is set as 999999, the specified value for the equipment is set as follows.
  Minimum display unit=0.01mm: 9999.99mm
  Minimum display unit=0.001mm: 999.999mm
  Minimum display unit=0.0001mm: 99.9999mm
  Minimum display unit=0.00001mm: 9.99999mm
  Minimum display unit=0.1µm: 99999.9µm
  Minimum display unit=0.1µm: 9999.99µm
