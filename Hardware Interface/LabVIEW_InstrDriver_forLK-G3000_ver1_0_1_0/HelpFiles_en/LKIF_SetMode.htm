Mode Switch (LKIF_SetMode)

BOOL WINAPI LKIF_SetMode(IN LKIF_MODE Mode);
Description Changes the operation mode of the equipment.
Parameter Mode Operation mode. The following values can be set.

typedef enum {
    LKIF_MODE_NORMAL, // normal mode
    LKIF_MODE_COMMUNICATION, // setting mode
} LKIF_MODE;
Returned Value TRUE Completes without errors
FALSE Completes with errors
Reference Running any API will automatically switch to operation mode, so there is no need to explicitly call this API.
