Get Calculation Method (LKIF_GetCalcMethod)

BOOL WINAPI LKIF_GetCalcMethod(IN int OutNo,OUT LKIF_CALCMETHOD *CalcMethod,LKIF_CALCTARGET *CalcTarget);
Description Returns the method for calculation.
Parameter OutNo The returned OUT number. Sets 0 (OUT1) or 1 (OUT2).
CalcMethod Returns the method for calculation.
CalcTarget Returns the measurement target.
Returned Value TRUE Completes without errors
FALSE Completes with errors
Reference Refer to Set Calculation Method for more about LKIF_CALCMETHOD and LKIF_CALCTARGET.
