ABLE Control Range (LKIF_GetAbleMinMax)

BOOL WINAPI LKIF_GetAbleMinMax(IN int HeadNo,OUT int *Min,OUT int *Max);
Description Returns the ABLE control range.
Parameter HeadNo Sets the returned head number. Sets 0 (HEAD-A) or 1 (HEAD-B).
Min Returns the minimum ABLE value.
Max Returns the maximum ABLE value.
Returned Value TRUE Completes without errors
FALSE Completes with errors
