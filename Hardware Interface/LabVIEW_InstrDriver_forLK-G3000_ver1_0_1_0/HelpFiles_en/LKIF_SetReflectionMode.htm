Set Mounting Mode (LKIF_SetReflectionMode)

BOOL WINAPI LKIF_SetReflectionMode(IN int HeadNo,IN LKIF_REFLECTIONMODE ReflectionMode);
Description Sets the mounting mode.
Parameter HeadNo Sets 0 (HEAD-A) or 1 (HEAD-B) as the head number.
ReflectionMode Mounting mode. The following values can be set.

typedef enum {
    LKIF_REFLECTIONMODE_DIFFUSION, // diffuse reflection
    LKIF_REFLECTIONMODE_MIRROR, // specular reflection
} LKIF_REFLECTIONMODE;
Returned Value TRUE Completes without errors
FALSE Completes with errors
