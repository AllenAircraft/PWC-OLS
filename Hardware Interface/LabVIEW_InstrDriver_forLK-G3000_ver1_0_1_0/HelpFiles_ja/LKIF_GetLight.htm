受光波形取得 (LKIF_GetLight)

　

BOOL WINAPI LKIF_GetLight(IN int HeadNo,IN int PeekNo,OUT int *MeasurePosition,OUT int *NumReaded,OUT BYTE *Value);
　
説明　　　　 受光波形を取得します。
　
パラメータ HeadNo 取得するHead番号。０（HEAD-A）、１（HEAD-B）を指定します。
PeekNo 測定モードが「透明体２」の時、波形を取得するピーク位置を指定します。０〜３の値を指定します。
MeasurePosition 測定位置を取得します。4個の配列を指定します。 受信データは０〜1024までの範囲の値で、０が無効データ、０以外が有効な測定位置のデータです。
NumReaded 読み出した有効データ数を取得します。
Value 読み出したデータを取得します。1024個の配列を指定します。
　
戻り値 TRUE 正常終了
FALSE 異常終了
