トリガモード設定 (LKIF_SetTriggerMode)

　

BOOL　WINAPI LKIF_SetTriggerMode(IN int OutNo,IN LKIF_TRIGGERMODE TriggerMode);
　
説明 トリガモードを設定します。
　
パラメータ OutNo 設定するOUT番号。０（OUT1）か１（OUT2）を指定します。
　 TriggerMode トリガモード。以下の値を指定します。

typedef enum {
　　LKIF_TRIGGERMODE_EXT1, // 外部トリガ１
　　LKIF_TRIGGERMODE_EXT2, // 外部トリガ２
} LKIF_TRIGGERMODE;
　
戻り値 TRUE 正常終了
FALSE 異常終了
