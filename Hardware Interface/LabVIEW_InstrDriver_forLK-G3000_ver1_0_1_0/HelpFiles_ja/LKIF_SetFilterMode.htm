フィルタモード設定 (LKIF_SetFilterMode)

　

BOOL WINAPI LKIF_SetFilterMode(IN int OutNo,OUT LKIF_FILTERMODE FilterMode);
　
説明 フィルタモードを設定します。
　
パラメータ OutNo 設定するOUT番号。０（OUT1）、１（OUT2）を指定します。
FilterMode フィルタモードを設定します。以下の値を設定します。

typedef enum {
　　LKIF_FILTERMODE_MOVING_AVERAGE, // 移動平均
　　LKIF_FILTERMODE_LOWPASS, // ローパスフィルタ
　　LKIF_FILTERMODE_HIGHPASS, // ハイパスフィルタ
} LKIF_FILTERMODE;
　
戻り値 TRUE 正常終了
FALSE 異常終了
