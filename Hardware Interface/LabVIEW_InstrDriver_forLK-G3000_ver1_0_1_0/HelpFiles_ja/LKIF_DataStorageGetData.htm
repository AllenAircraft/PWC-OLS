データストレージ出力 (LKIF_DataStorageGetData)

　

BOOL WINAPI LKIF_DataStorageGetData(IN int OutNo,IN int NumOutBuffer,OUT LKIF_FLOATVALUE *OutBuffer,OUT int *NumReceived);
　
説明 データストレージの蓄積データを取得します。
　
パラメータ OutNo 取得するOUT番号。０（OUT1)か１（OUT2)を指定します。
NumOutBuffer 取得するデータの個数を指定します。１〜６５５３６を指定します。
OutBuffer 蓄積データを取得します。
NumReceived 取得できた蓄積データの数を取得します。
　
戻り値 TRUE 正常終了
FALSE 異常終了
　
備考 データストレージ蓄積中はエラーリターンするので注意してください。
LKIF_FLOATVALUEについては、 測定値出力の説明をご覧ください。
