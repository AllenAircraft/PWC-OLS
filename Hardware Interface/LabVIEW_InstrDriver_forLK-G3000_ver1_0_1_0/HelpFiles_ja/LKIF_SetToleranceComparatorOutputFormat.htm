判定出力形態設定 (LKIF_SetToleranceComparatorOutputFormat)

　

BOOL WINAPI LKIF_SetToleranceComparatorOutputFormat(IN LKIF_TOLERANCE_COMPARATOR_OUTPUT_FORMAT ToleranceComparatorOutputFormat);
　
説明　　　　 判定出力形態を設定します。
　
パラメータ ToleranceComparetorOutputFormat 判定出力形態モード。以下の値から指定します。

typedef enum {
　　LKIF_TOLERANCE_COMPARATOR_OUTPUT_FORMAT_NORMAL, // ノーマル
　　LKIF_TOLERANCE_COMPARATOR_OUTPUT_FORMAT_HOLD, //ホールド
　　LKIF_TOLERANCE_COMPARATOR_OUTPUT_FORMAT_OFF_DELAY, // オフディレイ
} LKIF_TOLERANCE_COMPARATOR_OUTPUT_FORMAT;
　
　
戻り値 TRUE 正常終了
FALSE 異常終了
