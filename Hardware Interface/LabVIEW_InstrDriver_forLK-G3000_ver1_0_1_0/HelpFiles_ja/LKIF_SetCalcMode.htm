計測モード設定 (LKIF_SetCalcMode)

　

BOOL WINAPI LKIF_SetCalcMode(IN int OutNo,IN LKIF_CALCMODE CalcMode);
　
説明 計測モードを設定します。
　
パラメータ OutNo 設定するOUT番号。０（OUT1）か１（OUT2）を指定します。
CalcMode 測定モード。以下の値を指定します。

typedef enum {
　　LKIF_CALCMODE_NORMAL, // 標準
　　LKIF_CALCMODE_PEAKHOLD, //ピークホールド
　　LKIF_CALCMODE_BOTTOMHOLD, // ボトムホールド
　　LKIF_CALCMODE_PEAKTOPEAKHOLD, // ピークtoピークホールド
　　LKIF_CALCMODE_SAMPLEHOLD, // サンプルホールド
　　LKIF_CALCMODE_AVERAGEHOLD, // アベレージホールド
} LKIF_CALCMODE;
　
戻り値 TRUE 正常終了
FALSE 異常終了
