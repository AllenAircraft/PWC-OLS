統計結果出力 (LKIF_GetFigureData)

　

BOOL WINAPI LKIF_GetFigureData(IN int OutNo,OUT LKIF_FIGUREDATA *FigureData);
　
説明 統計結果を取得します。
　
パラメータ OutNo 取得するOUT番号。０（OUT1）、１（OUT2）を指定します。
FigureData 統計結果データを取得します。以下の構造体を指定します。

typedef struct {
　　float ToleUpper; // 公差上限値
　　float ToleLower; // 公差下限値
　　float AverageValue; // 平均値
　　float MaxValue; // 最大値
　　float MinValue; // 最小値
　　float DifValue; // 最大値 ? 最小値
　　float SdValue; // 標準偏差
　　LONG DataCnt; // 総データ数
　　LONG HighDataCnt; // High判定データ数
　　LONG GoDataCnt; // Go判定データ数
　　LONG LowDataCnt; // Low判定データ数
} LKIF_FIGUREDATA;
　
戻り値 TRUE 正常終了
FALSE 異常終了
