測定モード設定 (LKIF_SetMeasureMode)

　

BOOL WINAPI LKIF_SetMeasureMode(IN int HeadNo,IN LKIF_MEASUREMODE MeasureMode);
　
説明 測定モードを設定します。
　
パラメータ HeadNo 設定するヘッド番号を指定します。０（HEAD-A）か１（HEAD-B)を指定します。
MeasureMode 測定モード。以下の値を指定します。

typedef enum {
　　LKIF_MEASUREMODE_NORMAL, // 標準
　　LKIF_MEASUREMODE_HALF_T, // 半透明体
　　LKIF_MEASUREMDOE_TRAN_1, // 透明体
　　LKIF_MEASUREMODE_TRAN_2, // 透明体2
　　LKIF_MEASUREMODE_MRS, // 多重反射体
} LKIF_MEASUREMODE;
　
戻り値 TRUE 正常終了
FALSE 異常終了
