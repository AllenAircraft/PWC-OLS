演算方法設定 (LKIF_SetCalcMethod)

　

BOOL WINAPI LKIF_SetCalcMethod(IN int OutNo,IN LKIF_CALCMETHOD CalcMethod,LKIF_CALCTARGET CalcTarget);
　
説明 演算方法を設定します。
　
パラメータ OutNo 設定するOUT番号。０（OUT1）、１（OUT2）を指定します。
CalcMethod 演算方法。以下の値を指定します。

typedef enum {
　　LKIF_CALCMETHOD_HEADA, // ヘッドA
　　LKIF_CALCMETHOD_HEADB, // ヘッドB
　　LKIF_CALCMETHOD_HEAD_HEADA_PLUS_HEADB, // ヘッドA＋ヘッドB
　　LKIF_CALCMETHOD_HEAD_HEADA_MINUS_HEADB, // ヘッドA−ヘッドB
　　LKIF_CALCMETHOD_HEAD_HEADA_TRANSPARENT, // ヘッドA透明体
　　LKIF_CALCMETHOD_HEAD_HEADB_TRANSPARENT, // ヘッドB透明対
} LKIF_CALCMETHOD;
CalcTarget 測定対象。以下の値を指定します。

typedef enum {
　　LKIF_CALCTARGET_PEAK_1, // ピーク1
　　LKIF_CALCTARGET_PEAK_2, // ピーク2
　　LKIF_CALCTARGET_PEAK_3, // ピーク3
　　LKIF_CALCTARGET_PEAK_4, // ピーク4
　　LKIF_CALCTARGET_PEAK_1_2, // ピーク1−ピーク2
　　LKIF_CALCTARGET_PEAK_1_3, // ピーク1−ピーク3
　　LKIF_CALCTARGET_PEAK_1_4, // ピーク1−ピーク4
　　LKIF_CALCTARGET_PEAK_2_3, // ピーク2−ピーク3
　　LKIF_CALCTARGET_PEAK_2_4, // ピーク2−ピーク4
　　LKIF_CALCTARGET_PEAK_3_4, // ピーク3−ピーク4
} LKIF_CALCTARGET;
　
　
戻り値 TRUE 正常終了
FALSE 異常終了
