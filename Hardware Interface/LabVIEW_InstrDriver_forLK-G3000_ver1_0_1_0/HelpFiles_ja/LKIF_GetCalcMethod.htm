演算方法取得 (LKIF_GetCalcMethod)

　

BOOL WINAPI LKIF_GetCalcMethod(IN int OutNo,OUT LKIF_CALCMETHOD *CalcMethod,LKIF_CALCTARGET *CalcTarget);
　
説明 演算方法を取得します。
　
パラメータ OutNo 取得するOUT番号。０（OUT1）、１（OUT2）を指定します。
CalcMethod 演算方法を取得します。
CalcTarget 測定対象を取得します。
　
戻り値 TRUE 正常終了
FALSE 異常終了
　
備考 LKIF_CALCMETHOD,LKIF_CALCTARGETの詳細は、演算方法設定をご覧ください。
