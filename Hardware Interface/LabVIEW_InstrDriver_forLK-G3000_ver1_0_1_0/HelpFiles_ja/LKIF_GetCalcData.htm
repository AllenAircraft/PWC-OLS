測定値出力 (LKIF_GetCalcData)

　

BOOL WINAPI LKIF_GetCalcData(OUT LKIF_FLOATVALUE *CalcData1,OUT LKIF_FLOATVALUE *CalcData2);
　
説明 測定値を取得します。
　
パラメータ CalcData1 OUT1の測定値。以下の構造体に値が設定されます。
CalcData2 OUT2の測定値。以下の構造体に値が設定されます。

typedef enum {
　　LKIF_FLOATRESULT_VALID, // 有効データ
　　LKIF_FLOATRESULT_RANGEOVER_P, // +レンジオーバー
　　LKIF_FLOATRESULT_RANGEOVER_N, // −レンジオーバー
　　LKIF_FLOATRESULT_WAITING, // 判定待機
} LKIF_FLOATRESULT;


typedef struct {
　　LKIF_FLOATRESULT FloatResult; // 有効データかどうか
　　float Value; // LKIF_FLOATRESULT_VALIDの時測定値。それ以外の場合は無効値。
} LKIF_FLOATVALUE;
　
戻り値 TRUE 正常終了
FALSE 異常終了
　
備考 LKIF_FLOATVALUE構造体のValueに設定される値は、本体側の最小表示単位によって異なります。
例 ： 測定値を９９９９９９とした時、本体に設定される値を示します。

　最小表示単位＝０．１μｍ ： ９９９９９．９μｍ
　最小表示単位＝０．０１μｍ ： ９９９９．９９μｍ
