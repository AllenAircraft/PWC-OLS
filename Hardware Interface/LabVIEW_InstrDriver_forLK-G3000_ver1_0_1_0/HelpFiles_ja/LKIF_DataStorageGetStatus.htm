データストレージ蓄積状態出力 (LKIF_DataStorageGetStatus)

　

BOOL WINAPI LKIF_DataStorageGetStatus(IN int OutNo,OUT BOOL *IsStorage,OUT int *NumStorageData);
　
説明 データストレージの蓄積状態を取得します。
　
パラメータ OutNo 取得するOUT番号。０（OUT1)か１（OUT2)を指定します。
IsStorage 蓄積中かどうかを取得します。TRUE ： 蓄積中、FALSE ： 停止中
NumStorageData 蓄積されているデータの件数を取得します。０から６５５３６の値が返ります。
　
戻り値 TRUE 正常終了
FALSE 異常終了
