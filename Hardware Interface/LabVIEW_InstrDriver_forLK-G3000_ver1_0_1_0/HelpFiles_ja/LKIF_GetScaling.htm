スケーリング取得 (LKIF_GetScaling)

　

BOOL WINAPI LKIF_GetScaling(IN int OutNo,IN int HeadNo,OUT int *InputValue1,OUT int *OutputValue1,OUT int *InputValue2,OUT int *OutputValue2);
　
説明 スケーリングを取得します。
　
パラメータ OutNo 設定するOUT番号。０（OUT1）、１（OUT2）を指定します。
HeadNo ヘッド番号０（HEAD-A）か１（HEAD-B)を指定します。
InputValue1 入力値１を取得します。-999999〜99999の値を取得します。
OutputValue1 表示値１を取得します。-999999〜99999の値を取得します。
InputValue2 入力値２を取得します。-999999〜99999の値を取得します。
OutputValue2 表示値２を取得します。-999999〜99999の値を取得します。
　
戻り値 TRUE 正常終了
FALSE 異常終了
　
備考 本体に設定される値は、本体側の最小表示単位によって異なります。
例 ： 公差を９９９９９９とした時、本体に設定される値を示します。
　最小表示単位＝０．０１ｍｍ ： ９９９９．９９ｍｍ
　最小表示単位＝０．００１ｍｍ ： ９９９．９９９ｍｍ
　最小表示単位＝０．０００１ｍｍ ： ９９．９９９９ｍｍ
　最小表示単位＝０．００００１ｍｍ ： ９．９９９９９ｍｍ
　最小表示単位＝０．１μｍ ： ９９９９９．９μｍ
　最小表示単位＝０．０１μｍ ： ９９９９．９９μｍ
