平均回数設定 (LKIF_SetAverage)

　

BOOL WINAPI LKIF_SetAverage(IN int OutNo,IN LKIF_AVERAGE Average);
　
説明 平均回数を設定します。
　
パラメータ OutNo 設定するOUT番号。０（OUT1)か１（OUT2)を指定します。
Average 平均回数。以下の値を指定します。

typedef enum {
　　LKIF_AVERAGE_1, // 1回
　　LKIF_AVERAGE_4, //
　　LKIF_AVERAGE_16, //
　　LKIF_AVERAGE_64, //
　　LKIF_AVERAGE_256, //
　　LKIF_AVERAGE_1024, //
　　LKIF_AVERAGE_4096, //
　　LKIF_AVERAGE_16384, //
　　LKIF_AVERAGE_65536, //
　　LKIF_AVERAGE_262144, // 262144回
} LKIF_AVERAGE;
　
戻り値 TRUE 正常終了
FALSE 異常終了
