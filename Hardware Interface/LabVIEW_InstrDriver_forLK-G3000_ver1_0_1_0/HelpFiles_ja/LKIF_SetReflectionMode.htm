設置モード設定 (LKIF_SetReflectionMode)

　

BOOL WINAPI LKIF_SetReflectionMode(IN int HeadNo,IN LKIF_REFLECTIONMODE ReflectionMode);
　
説明 設置モードを設定します。
　
パラメータ HeadNo ヘッド番号０（HEAD-A）か１（HEAD-B)を指定します。
ReflectionMode 設置モード。以下の値を指定します。

typedef enum {
　　LKIF_REFLECTIONMODE_DIFFUSION, // 拡散反射
　　LKIF_REFLECTIONMODE_MIRROR, // 正反射
} LKIF_REFLECTIONMODE;
　
戻り値 TRUE 正常終了
FALSE 異常終了
