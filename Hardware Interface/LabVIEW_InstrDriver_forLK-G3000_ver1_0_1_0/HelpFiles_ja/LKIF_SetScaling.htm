スケーリング設定 (LKIF_SetScaling)

　

BOOL WINAPI LKIF_SetScaling(IN int OutNo,IN int HeadNo,IN int InputValue1,IN int OutputValue1,IN int InputValue2,IN int OutputValue2);
　
説明 スケーリングを設定します。
　
パラメータ OutNo 設定するOUT番号。０（OUT1)か１（OUT2)を指定します。
HeadNo 設定するヘッド番号。０（HEAD-A）か１（HEAD-B)を指定します。
InputValue1、OutputValue1 入力値→表示値の値を指定します。-999999から999999の値を指定します。
InputValue2、OutputValue2 入力値→表示値の値を指定します。-999999から999999の値を指定します。
　
戻り値 TRUE 正常終了
FALSE 異常終了
　
備考 入力値と出力値のパラメータは以下の範囲で設定します。

スパン = (表示値１-表示値２) / (入力値１-入力値２)
-9.999<=スパン<=9.999
　

本体に設定される値は、本体側の最小表示単位によって異なります。
例 ： スケーリングを９９９９９９とした時、本体に設定される値を示します。
　最小表示単位＝０．０１ｍｍ ： ９９９９．９９ｍｍ
　最小表示単位＝０．００１ｍｍ ： ９９９．９９９ｍｍ
　最小表示単位＝０．０００１ｍｍ ： ９９．９９９９ｍｍ
　最小表示単位＝０．００００１ｍｍ ： ９．９９９９９ｍｍ
　最小表示単位＝０．１μｍ ： ９９９９９．９μｍ
　最小表示単位＝０．０１μｍ ： ９９９９．９９μｍ
