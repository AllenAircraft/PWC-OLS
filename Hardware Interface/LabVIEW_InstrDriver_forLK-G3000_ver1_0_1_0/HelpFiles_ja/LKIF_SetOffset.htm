オフセット設定 (LKIF_SetOffset)

　

BOOL WINAPI LKIF_SetOffset(IN int OutNo,IN int Offset);
　
説明 オフセットを設定します。
　
パラメータ OutNo 設定するOUT番号。０（OUT1)か１（OUT2)を指定します。
Offset オフセット。-999999〜999999の値を指定します。
　
戻り値 TRUE 正常終了
FALSE 異常終了
　
備考 本体に設定される値は、本体側の最小表示単位によって異なります。
例 ： オフセットを９９９９９９とした時、本体に設定される値を示します。
　最小表示単位＝０．０１ｍｍ ： ９９９９．９９ｍｍ
　最小表示単位＝０．００１ｍｍ ： ９９９．９９９ｍｍ
　最小表示単位＝０．０００１ｍｍ ： ９９．９９９９ｍｍ
　最小表示単位＝０．００００１ｍｍ ： ９．９９９９９ｍｍ
　最小表示単位＝０．１μｍ ： ９９９９９．９μｍ
　最小表示単位＝０．０１μｍ ： ９９９９．９９μｍ
