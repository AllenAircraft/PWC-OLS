データストレージ設定 (LKIF_SetDataStorage)

　

BOOL WINAPI LKIF_SetDataStorage(IN LKIF_TARGETOUT TargetOut,,IN int NumStorage,IN LKIF_STORAGECYCLE StorageCycle);
　
説明 データストレージの対象OUT、蓄積点数、蓄積周期を設定します。
　
パラメータ TargetOut 対象OUT。以下の値を指定します。

typedef enum {
　　LKIF_TARGETOUT_NONE, // 対象OUTなし
　　LKIF_TARGETOUT_OUT1, // OUT１
　　LKIF_TARGETOUT_OUT2, // OUT２
　　LKIF_TARGETOUT_BOTH, // OUT1とOUT2
} LKIF_TARGETOUT;
NumStorage 蓄積点数を指定します。１〜６５５３６の値を指定します。
StorageCycle 蓄積周期。以下の値を指定します。

typedef enum {
　　LKIF_STORAGECYCLE_1, // サンプリング周期×１
　　LKIF_STORAGECYCLE_2, //サンプリング周期×２
　　LKIF_STORAGECYCLE_5, // サンプリング周期×５
　　LKIF_STORAGECYCLE_10, // サンプリング周期×１０
　　LKIF_STORAGECYCLE_20, // サンプリング周期×２０
　　LKIF_STORAGECYCLE_50, // サンプリング周期×５０
　　LKIF_STORAGECYCLE_100, // サンプリング周期×１００
　　LKIF_STORAGECYCLE_200, // サンプリング周期×２００
　　LKIF_STORAGECYCLE_500, // サンプリング周期×５００
　　LKIF_STORAGECYCLE_1000, // サンプリング周期×１０００
} LKIF_STORAGECYCLE;
　
戻り値 TRUE 正常終了
FALSE 異常終了
