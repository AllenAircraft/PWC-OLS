アナログ出力スケーリング設定 (LKIF_SetAnalogScaling)

　

BOOL WINAPI LKIF_SetAnalogScaling(IN int OutNo,IN int InputValue1,IN int OutputVoltage1,IN int InputValue2,IN int OutputVoltage2);
　
説明 アナログ出力スケーリングを設定します。
　
パラメータ OutNo 設定するOUT番号。０（OUT1）、１（OUT2）を指定します。
InputValue1 測定値１を指定します。-999999〜999999の値を指定します。
OutputVoltage1 出力電圧１を指定します。-10500(mV)〜10500(mV)の値を指定します。(単位は固定)
InputValue2 測定値２を指定します。-999999〜999999の値を指定します。
OutputVoltage2 出力電圧２を指定します。-10500(mV)〜10500(mV)の値を指定します。(単位は固定)
　
戻り値 TRUE 正常終了
FALSE 異常終了
　
備考 本体に設定される値は、本体側の最小表示単位によって異なります。
例 ： 測定値１を９９９９９９とした時、本体に設定される値を示します。
　最小表示単位＝０．０１ｍｍ ： ９９９９．９９ｍｍ
　最小表示単位＝０．００１ｍｍ ： ９９９．９９９ｍｍ
　最小表示単位＝０．０００１ｍｍ ： ９９．９９９９ｍｍ
　最小表示単位＝０．００００１ｍｍ ： ９．９９９９９ｍｍ
　最小表示単位＝０．１μｍ ： ９９９９９．９μｍ
　最小表示単位＝０．０１μｍ ： ９９９９．９９μｍ
