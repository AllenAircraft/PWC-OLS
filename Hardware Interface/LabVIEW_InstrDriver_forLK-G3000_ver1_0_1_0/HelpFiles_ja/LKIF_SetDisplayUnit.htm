最小表示単位設定 (LKIF_SetDisplayUnit)

　

BOOL WINAPI LKIF_SetDisplayUnit(IN int OutNo,IN LKIF_DISPLAYUNIT DisplayUnit);
　
説明

最小表示単位を設定します。
　
パラメータ OutNo 設定するOUT番号。０（OUT1）、１（OUT2）を指定します。
DisplayUnit 最小表示単位。以下の値を指定します。

typedef enum {
　　LKIF_DISPLAYUNIT_0000_01MM, // 0.01mm
　　LKIF_DISPLAYUNIT_000_001MM, // 0.001mm
　　LKIF_DISPLAYUNIT_00_0001MM, // 0.0001mm
　　LKIF_DISPLAYUNIT_0_00001MM, // 0.00001mm
　　LKIF_DISPLAYUNIT_0000_01UM, // 0.01μm
　　LKIF_DISPLAYUNIT_000_001UM, // 0.001μm
} LKIF_DISPLAYUNIT;
　
戻り値 TRUE 正常終了
FALSE 異常終了
