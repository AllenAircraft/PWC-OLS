カットオフ周波数設定 (LKIF_SetCutOffFrequency)

　

BOOL WINAPI LKIF_SetCutOffFrequency(IN int OutNo,IN LKIF_CUTOFFFREQUENCY CutOffFrequency);
　
説明 フィルタ種別がローパス／ハイパス設定時のカットオフ周波数を設定します。
　
パラメータ OutNo 設定するOUT番号。０（OUT1）、１（OUT2）を指定します。
CutOffFrequency カットオフ周波数を指定します。以下の値を指定します。

typedef enum {
　　LKIF_CUTOFFFREQUENCY_1000, // １０００Ｈｚ
　　LKIF_CUTOFFFREQUENCY_300, // ３００Ｈｚ
　　LKIF_CUTOFFFREQUENCY_100, // １００Ｈｚ
　　LKIF_CUTOFFFREQUENCY_30, // ３０Ｈｚ
　　LKIF_CUTOFFFREQUENCY_10, // １０Ｈｚ
　　LKIF_CUTOFFFREQUENCY_3, // ３Ｈｚ
　　LKIF_CUTOFFFREQUENCY_1, // １Ｈｚ
　　LKIF_CUTOFFFREQUENCY_0_3, // ０．３Ｈｚ
　　LKIF_CUTOFFFREQUENCY_0_1, // ０．１Ｈｚ
} LKIF_CUTOFFFREQUENCY;
　
戻り値 TRUE 正常終了
FALSE 異常終了
