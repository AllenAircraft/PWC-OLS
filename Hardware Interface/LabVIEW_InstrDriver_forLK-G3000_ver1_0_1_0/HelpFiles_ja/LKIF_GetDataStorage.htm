データストレージ取得 (LKIF_GetDataStorage)

　

BOOL WINAPI LKIF_GetDataStorage(OUT LKIF_TARGETOUT *TargetOut,OUT int *NumStorage,OUT LKIF_STORAGECYCLE *StorageCycle);
　
説明 データストレージ設定の対象OUT、蓄積点数、蓄積周期を取得します。
　
パラメータ TargetOut 対象OUTを取得します。
NumStorage 蓄積点数を取得します。
StorageCycle 蓄積周期を取得します。
　
戻り値 TRUE 正常終了
FALSE 異常終了
　
備考 LKIF_TARGETOUT、LKIF_STORAGECYCLEの詳細は、データストレージ設定をご覧ください。
