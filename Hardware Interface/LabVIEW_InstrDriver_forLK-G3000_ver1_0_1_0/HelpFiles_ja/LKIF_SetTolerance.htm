公差設定 (LKIF_SetTolerance)

　

BOOL WINAPI LKIF_SetTolerance(IN int OutNo,IN int UpperLimit,IN int LowerLimit,IN int Hysteresis);
　
説明 公差を設定します。
　
パラメータ OutNo 設定するOUT番号。０（OUT1）か１（OUT2）を指定します。
UpperLimit 公差上限値を指定します。-999999から999999の値を指定します。
LowerLimit 公差下限値を指定します。-999999から999999の値を指定します。
Hysteresis 公差ヒステリシスの値を指定します。0から999999の値を指定します。
　
戻り値 TRUE 正常終了
FALSE 異常終了
　
備考 本体に設定される値は、本体側の最小表示単位によって異なります。
例 ： 公差を９９９９９９とした時、本体に設定される値を示します。
　最小表示単位＝０．０１ｍｍ ： ９９９９．９９ｍｍ
　最小表示単位＝０．００１ｍｍ ： ９９９．９９９ｍｍ
　最小表示単位＝０．０００１ｍｍ ： ９９．９９９９ｍｍ
　最小表示単位＝０．００００１ｍｍ ： ９．９９９９９ｍｍ
　最小表示単位＝０．１μｍ ： ９９９９９．９μｍ
　最小表示単位＝０．０１μｍ ： ９９９９．９９μｍ
