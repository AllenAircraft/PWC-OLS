アナログ出力スケーリング取得 (LKIF_GetAnalogScaling)

　

BOOL WINAPI LKIF_GetAnalogScaling(IN int OutNo,OUT int *InputValue1,OUT int *OutputVoltage1,OUT int *InputValue2,OUT int *OutputVoltage2);
　
説明 アナログ出力スケーリングを取得します。
　
パラメータ OutNo 取得するOUT番号。０（OUT1）、１（OUT2）を指定します。
InputValue1 測定値１を取得します。-999999〜999999の値を取得します。
OutputVoltage1 出力電圧１を取得します。-10500(mV)〜10500(mV)の値を取得します。(単位は固定)
InputValue2 測定値２を取得します。-999999〜999999の値を取得します。
OutputVoltage2 出力電圧２を取得します。-10500(mV)〜10500(mV)の値を取得します。(単位は固定)
　
戻り値 TRUE 正常終了
FALSE 異常終了
備考 本体に設定される値は、本体側の最小表示単位によって異なります。
例 ： 測定値１を９９９９９９とした時、本体に設定される値を示します。
　最小表示単位＝０．０１ｍｍ ： ９９９９．９９ｍｍ
　最小表示単位＝０．００１ｍｍ ： ９９９．９９９ｍｍ
　最小表示単位＝０．０００１ｍｍ ： ９９．９９９９ｍｍ
　最小表示単位＝０．００００１ｍｍ ： ９．９９９９９ｍｍ
　最小表示単位＝０．１μｍ ： ９９９９９．９μｍ
　最小表示単位＝０．０１μｍ ： ９９９９．９９μｍ

出力電圧は、mV単位の値を返します。
